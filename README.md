## Project Overview

This Tableau dashboard analyzes financial data by visualizing outstanding balances, interest rates, and client performance. It provides interactive insights into outstanding balance distribution, correlation analysis, and the top 5 clients based on outstanding balances over time.

---

## Objectives

- Analyze outstanding balances across different interest rates.
- Identify the relationship between interest rates and outstanding balances.
- Visualize the Top 5 clients with the highest outstanding balances over time.
- Provide interactive filtering using Reporting Date.

---

## Dataset

The dataset includes:

- Client Name
- Client Code
- Product Description
- Interest Rate
- Balance Outstanding
- Reporting Date

---

## Dashboard Features

- 📊 Outstanding Balance Distribution
- 📈 Correlation Analysis with Trend Line
- 👥 Top 5 Clients Over Time
- 📅 Interactive Reporting Date Filter
